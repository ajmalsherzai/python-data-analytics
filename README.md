# Python Data Analytics Project

## Project Overview
This project demonstrates data analytics using Python. The goal is to clean, analyze, and visualize business data using common Python libraries such as Pandas, NumPy, and Matplotlib.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Data Cleaning
- Exploratory Data Analysis
- Business Reporting

## Skills Demonstrated
- Creating and working with datasets
- Cleaning missing or inconsistent data
- Calculating summary statistics
- Grouping and aggregating data
- Creating charts
- Finding business insights
- Communicating results clearly

## Business Questions Answered
- Which product category generated the most revenue?
- Which region had the strongest sales?
- What was the average order value?
- Which customers or products performed best?
- What trends can be found from the data?

## Files
- `python_data_analysis.py` - Python script containing data creation, cleaning, analysis, visualization, and business insights.
